RAG — Retrieval-Augmented

Base de Conhecimento Vetorial

Indexação FAISS com OpenAI Embeddings. Cada consulta recupera os 2 documentos mais relevantes em milissegundos — seja sobre reset de senha, Wi-Fi WPA2 ou portal acadêmico.

FAISS Embeddings similarity_search(k=2)
LangGraph — ReAct Agent

Raciocínio + Ação Nativo

O agente ReAct decide qual ferramenta acionar, executa, observa e raciocina novamente — com diretivas de clarificação obrigatórias para ambiguidades. Nenhuma ação precipitada.

create_react_agent Tool Calling gpt-4o-mini
Tools — Diagnóstico Ativo

APIs de Diagnóstico em Tempo Real

Verificação de credenciais via mock_auth_api e status de roteadores via mock_network_status. Diagnóstico determinístico, sem suposições.

mock_auth_api mock_network_status @tool
Arquitetura do Sistema

Modular.
Determinístico.
Pronto para Produção.

Três camadas bem definidas: Django como orquestrador seguro de rotas, LangGraph como motor de raciocínio iterativo, e FAISS como memória vetorizada de longo prazo.

Frontend Minimalista (Nielsen)
HTML5 + Tailwind CSS. Interface limpa reduz carga cognitiva em momentos de falha técnica. Heurística de Estética e Design Minimalista.
Backend Django 6.0
Monolito Python com rotas seguras. Separa a camada de apresentação da lógica de negócio e do agente IA.
Core IA: LangGraph + FAISS
ReAct agent com 3 ferramentas integradas. FAISS indexa manuais e políticas. Temperature=0 garante determinismo.

Heurística #8 · Nielsen

Estética Minimalista

Em falhas técnicas o usuário já enfrenta alta carga cognitiva. Cores neutras e interfaces limpas removem o ruído visual e permitem foco exclusivo na solução do problema.

🛡️
Heurística #5 · Nielsen

Prevenção de Erros

Diretivas de Clarificação forçam alinhamento antes de qualquer ação. Evita resetar senhas incorretas ou reiniciar serviços críticos baseado em premissas falsas.

🎮
Heurística #3 · Nielsen

Controle e Liberdade

O sistema guia sem decidir autonomamente. Transforma o suporte — usualmente opaco e frustrante — em um fluxo transparente, colaborativo e previsível.
